Join the numbers and get to the 2048 tile!

How to play: Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one!

Note: This site is an Elm clone of the original 2048 JavaScript game that was created by Gabriele Cirulli.

Developed by Dwayne Crooks. 2048 is based on 1024 by Veewo Studio and conceptually similar to Threes by Asher Vollmer.
